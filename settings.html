⚙️ AI 功能设置

配置 AI API 以启用自动优化标题、生成摘要等功能

💡 使用说明

  • 支持所有兼容 OpenAI 格式的 API（中转API、DeepSeek、Kimi等）
  • 配置后可在采集页面使用"AI优化"按钮自动生成标题和摘要
  • API 密钥保存在本地，不会上传到任何服务器

🔌 API 配置

OpenAI兼容的API基础地址（结尾不要加斜杠）
你的 API 密钥，安全保存在本地
例如：gpt-3.5-turbo, gpt-4, deepseek-chat, moonshot-v1-8k

📣 公众号 API 配置

公众号 API 基础地址，默认 https://wx.limyai.com
后台请求公众号 API 的密钥
在“获取公众号列表”接口获取
用于将本地图片路径转换成公网可访问地址

📕 小红书发布配置

用于执行 xhs-auto-suite 的 Python 命令
xhs-auto-suite 的根目录
对应浏览器 profile 名称，默认 default
本地媒体路径会按该域名重写为公网地址
发布时调用本地小红书爆款标题 skill，仅改发布标题，不改收藏库原文
用于统一小红书搜索标签
开启后不会立即发布，系统会自动计算发布时间并提交到小红书平台；提交成功后即使你关机也会按平台时间发布
每条之间会在最小值和最大值之间随机取一个时间间隔，并自动填到小红书平台定时发布
