🎨 Chinese Prompt

大発見！！ Lovartのniji・journeyでも--srefコマンド使えます！ 🔥
sref（スタイルリファレンス）
＝画風を指定できるコード
Midjourneyはプロンプトにコマンド入力でサイズやクオリティを調整できるけど、srefはとにかく効果が絶大！
世界観が一気に統一できる強力なコマンドです✨
ちなみにsrefコードは約43億通り存在するって記述を見たのですが・・・ほんと？？（具体的には、コードの範囲は 0〜4,294,967,295 で正確には4,294,967,296通り）いつかお気に入りsref図鑑にしたいです！

すぐに使えるコードいくつか紹介しますね！
①ピンク×ブルーのアウトラインなしキュートスタイル
--sref 334246522

🎨 English Prompt

Big discovery!! You can use the --sref command even in Lovart’s niji・journey! 🔥
sref (Style Reference)
= a code that lets you specify the art style
In Midjourney you can adjust size and quality by entering commands in the prompt, but sref is incredibly effective!
It’s a powerful command that can instantly unify the overall aesthetic ✨
By the way, I saw a note that there are about 4.3 billion possible sref codes... is that true?? (Specifically, the code range is 0–4,294,967,295, so there are exactly 4,294,967,296 possibilities.) Someday I want to make a catalog of my favorite srefs!

Here are a few codes you can use right away!
