## 🚀 安装浏览器扩展（推荐使用）

### Chrome / Edge 浏览器

1. **打开扩展管理页面**
   - Chrome: 地址栏输入 `chrome://extensions/`
   - Edge: 地址栏输入 `edge://extensions/`

2. **开启开发者模式**
   - 在右上角找到"开发者模式"开关
   - 点击开启

3. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择文件夹：`E:\gitpromts\extension`
   - 点击"选择文件夹"

4. **完成！**
   - 扩展图标会出现在工具栏
   - 打开 X(Twitter) 就能看到采集按钮

---

## 📖 使用方法

### 方法1：使用浏览器扩展（推荐）

1. **安装扩展**（只需一次）
   按照上面的步骤安装

2. **打开X(Twitter)**
   访问 https://x.com 或 https://twitter.com

3. **浏览推文**
   每条推文下方会自动出现 **📚 采集** 按钮

4. **点击采集**
   - 点击"📚 采集"按钮
   - 按钮会变成"✅ 已采集"
   - 自动弹出编辑页面

5. **编辑保存**
   - 系统自动下载图片和视频
   - 填写或编辑信息
   - 点击保存

6. **生成数据**
   ```bash
   node scripts\generate-dataset.js
   ```

7. **查看效果**
   刷新 http://localhost:3000

---

### 方法2：使用书签工具

1. **创建书签**
   - 打开 `E:\gitpromts\bookmarklet-final-with-video.js`
   - 复制全部内容
   - 在书签栏新建书签
   - 将代码粘贴到URL位置

2. **使用**
   - 打开推文详情页
   - 点击书签
   - 自动采集

## ❓ 常见问题

### Q1: 扩展安装后看不到按钮？

**A:** 
1. 检查扩展是否已启用
2. 刷新X页面（F5）
3. 查看浏览器控制台是否有错误

### Q2: 采集按钮显示但点击无效？

**A:**
1. 确保服务器正在运行
2. 打开 http://localhost:3000 测试
3. 查看浏览器控制台的日志

### Q3: 只想用书签工具可以吗？

**A:** 当然可以！两种方式功能一样，选你喜欢的即可。

### Q4: 扩展会采集我的隐私数据吗？

**A:** 
不会！代码完全开源：
- 只在你点击"采集"时提取推文数据
- 数据只发送到本地服务器（localhost:3000）
- 不连接任何第三方服务器
