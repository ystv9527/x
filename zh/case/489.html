🎨 中文提示词

生成一张图像，展示该动物被简化并变形为动漫风格的毛绒玩具（材质为短绒、触感柔软的聚酯针织布），放置在 UFO 捞娃娃机中。
两侧各有一台额外的 UFO 捞娃娃机，里面摆放着不同动物的毛绒玩具，与主图像的动物不同。
场景设定在日本游戏中心，整体明亮。UFO 捞娃娃机的上部涂有鲜艳颜色，下部涂为白色。背景为墙面，UFO 捞娃娃机背后的区域模糊处理。地面铺有地毯。
拍摄角度为正面。最重要的是，图像中绝对不要出现任何文字或标志。

Create an image depicting an animal that has been simplified and stylized into a plush toy in anime style (made of short-pile, soft-touch polyester knitted fabric), placed inside a UFO claw machine. On each side, there are additional UFO claw machines containing different animal plush toys, which are different from the main animal. The scene is set in a Japanese arcade, with a bright overall lighting. The top of the UFO claw machine is painted in vibrant colors, while the bottom is white. The background features a wall, with the area behind the claw machine blurred. The floor is carpeted. The camera angle is frontal. Most importantly, the image must contain no text or logos whatsoever.
